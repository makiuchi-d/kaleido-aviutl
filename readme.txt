      万華鏡 フィルタプラグイン ver0.02  by MakKi
-----------------------------------------------------------------------

【機能】

　万華鏡エフェクトを加えます。

【インストール】

　kaleidoscope.auf を AviUtl の認識可能なフォルダに入れてください

【使用方法】

  ○パラメータ

  ・x, y スライダ
        基点となる三角形の頂点の位置をピクセル単位で指定します

  ・size (サイズ) スライダ
        三角形のサイズを指定します

  ・angle (傾き) スライダ
        三角形の傾きを指定します

  ・show tryangle (三角形を表示) チェックボックス
        基準の三角形の位置を表示します。


  ○フィルタ名、パラメタ名、スライダの最大最小初期値の変更方法

    フィルタのファイル名+.ini(デフォルトではkaleidoscope.auf.ini)という
    ファイルを作成し、フィルタ名とトラックバー、チェックボックスの
    名称、値を変更できます。
    詳しくは同梱されているkaleidoscope.auf.iniをご覧ください。
